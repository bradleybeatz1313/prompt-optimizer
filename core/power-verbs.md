# Power Verb Toolkit

Verb choice is the single fastest optimization. Action verbs activate the model's relevant training data and processing pathways for that specific task.

## The Problem

Weak verbs produce vague outputs:
- "Think about summarizing this" → ambiguous intent
- "Help me with a report" → unclear deliverable
- "Can you look at this code?" → no specific action

## The Solution

Replace with the verb that most precisely matches the desired action.

## Verb Categories

### Analysis & Evaluation
| Verb | Use When |
|------|----------|
| **Analyze** | Deep examination of components and relationships |
| **Evaluate** | Judging quality, effectiveness, or suitability |
| **Compare** | Side-by-side examination of two or more items |
| **Contrast** | Highlighting differences between items |
| **Measure** | Quantifying a specific metric or dimension |
| **Assess** | Broader evaluation with judgment |

### Discovery & Extraction
| Verb | Use When |
|------|----------|
| **Extract** | Pulling specific data from a larger body |
| **Identify** | Finding and naming specific elements |
| **Find** | Locating something within a search space |
| **Parse** | Breaking structured data into components |
| **Retrieve** | Fetching specific known information |
| **Detect** | Finding something that may or may not exist |

### Organization & Structure
| Verb | Use When |
|------|----------|
| **Categorize** | Sorting items into predefined groups |
| **Classify** | Assigning items to categories with reasoning |
| **Organize** | Arranging items into a logical structure |
| **Sort** | Ordering items by a specific criterion |
| **Rank** | Ordering items by relative quality or importance |
| **Group** | Clustering related items together |

### Creation & Transformation
| Verb | Use When |
|------|----------|
| **Create** | Producing something new from scratch |
| **Generate** | Producing multiple options or variations |
| **Write** | Producing prose or structured text |
| **Design** | Creating with intention around form and function |
| **Build** | Constructing something with components |
| **Rewrite** | Transforming existing text with new framing |
| **Translate** | Converting between languages or formats |
| **Summarize** | Condensing while preserving key information |

### Selection & Decision
| Verb | Use When |
|------|----------|
| **Select** | Choosing from a defined set of options |
| **Pick** | Quick selection with clear criteria |
| **Recommend** | Suggesting with reasoning and context |
| **Predict** | Forecasting based on available data |
| **Decide** | Making a judgment call with tradeoffs |

### Presentation & Output
| Verb | Use When |
|------|----------|
| **List** | Enumerating items (with or without detail) |
| **Describe** | Providing detailed characterization |
| **Define** | Providing precise meaning or specification |
| **Show** | Demonstrating with examples or visuals |
| **Explain** | Making something understandable |
| **Provide** | Delivering requested information |
| **Return** | Outputting in a specific format |

## Transformation Examples

| Before (Weak) | After (Strong) | Why Better |
|---------------|---------------|------------|
| "Think about this data" | "Analyze the trends in this dataset" | Specific action + target |
| "Help with my code" | "Debug this function and identify the root cause" | Two precise actions |
| "Look at these options" | "Rank these options by cost-effectiveness" | Clear criterion |
| "Do something with this text" | "Summarize this text in 3 bullet points" | Action + format + constraint |
| "Can you check this?" | "Evaluate this proposal against our requirements" | Action + evaluation framework |
