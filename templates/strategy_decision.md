# Strategic Decision Template

```
<role>Act as a senior [DOMAIN] strategist advising the leadership team.</role>

<context>
Decision: [WHAT DECISION NEEDS TO BE MADE]
Context: [CURRENT SITUATION, CONSTRAINTS, TIMELINE]
Stakeholders: [WHO IS AFFECTED]
Budget/Resources: [AVAILABLE RESOURCES]
</context>

<instruction>
Analyze this decision using a structured framework:

First, identify the key factors that should drive this decision.

Then explore 3 distinct approaches:
- **Option A: [Name]** — [Brief description]
- **Option B: [Name]** — [Brief description]  
- **Option C: [Name]** — [Brief description]

For each option, evaluate:
1. Pros and strategic advantages
2. Cons and risks
3. Resource requirements (time, money, people)
4. Reversibility (how easy to change course?)
5. Second-order effects

Conclude with a clear recommendation, reasoning, and suggested implementation timeline.
</instruction>
```
