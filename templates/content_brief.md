# Content Brief Template

```
<role>Act as a content strategist creating a detailed writing brief.</role>

<context>
Topic: [TOPIC]
Target audience: [AUDIENCE DESCRIPTION]
Content type: [blog post / whitepaper / social post / email / etc.]
Goal: [What should the reader do/feel/know after reading?]
Brand voice: [TONE DESCRIPTION]
</context>

<instruction>
Create a comprehensive content brief including:

1. **Working title** — 3 options ranked by click potential
2. **Target keywords** — Primary and secondary
3. **Outline** — Section-by-section with key points per section
4. **Word count target** — Based on content type and audience
5. **Key sources** — Types of evidence or examples to include
6. **Call to action** — What the reader should do next
7. **Internal links** — Suggested connections to existing content
8. **Visual needs** — Images, diagrams, or charts to include

Keep the brief actionable — a writer should be able to produce the content from this alone.
</instruction>
```
