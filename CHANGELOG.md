# Changelog

## [2.1.0] — 2026-05-15 (Desktop)

### Added
- APE (Automatic Prompt Engineering) technique
- Pydantic validation pattern for structured output
- Many-shot prompting guidance (10-100+ examples)
- Multimodal prompting section
- Code prompting patterns (generation, explanation, translation, debugging)
- Benchmark suite with 3 datasets (code, writing, analysis)
- Evaluation framework with heuristic scoring
- 6 reusable prompt templates

### Changed
- Expanded context engineering to full 4-layer model with implementation guidance
- Improved technique selection with decision matrix

## [2.0.0] — 2026-03-10 (Desktop + Code)

### Added
- Claude Code edition (streamlined for terminal workflows)
- Context engineering with 4-layer model
- RAG awareness technique
- Iterative framing for creative/design tasks
- Anti-patterns documentation
- Power verb reference card
- Migration guide from v1.x

### Changed
- Lowered activation threshold to any substantive request
- Simplified output format (removed technique list)
- Streamlined technique selection
- Reorganized into 4 clear stages

### Removed
- Complexity scoring (replaced by light/medium/heavy)
- Technique probability matrix (replaced by judgment-based selection)
